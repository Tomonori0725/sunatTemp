新規テーマの使い方

config.json

テーマの設定ファイルはconfig.jsonであり、中身はJSONになっています。
JSONとはデータの記法の1つで、取得したデータや設定などをプログラムに伝えるのに使われます。
InstagramやFacebook、ブログなどから取得した投稿データや、カルーセルのJavaScriptライブラリの設定などもJSONで、フロントエンドにも欠かせないものになっています。
「記法」なので書き方が厳密に決まっています。
例えばHTMLでもタグは < と > で囲まなければならない、<div>や<p>は閉じタグがあるけど<input>や<img>にはない、など、決まりがあります。
同じようにJSONにも決まりがあり、それに則って書かないとエラーになります。
ここでは、設定ファイルの中身に入る前にJSONの記法を簡単に説明します。

JSONを見るとまず { } [ ] : " , などの記号が目に入ると思います。
